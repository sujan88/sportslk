A Simple Shopping Cart

This will allow your visitors to choose from a list of priced items. The script will tally the items, add the tax and send the order to you. This demonstration uses a "mailto:" form with to a dummy address of "me@mydomain.com". If you click the "Send Order" button the order should be returned to you as undeliverable, so you will be able to see the way it works. Note: This type of form is only supported by Netscape and Internet Explorer 4.x. In a real order form, a cgi script would usually be used.
